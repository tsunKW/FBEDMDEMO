──────────────────────────

操作:
  1. 点击 "Generate new token"
  2. 选择 "Generate new token (classic)"
  3. 勾选权限: ✓ repo, ✓ workflow
  4. 点击 "Generate token"
  5. 复制显示的 Token

  2. 输入: cmd
  3. 按 Enter

方法 2 - 从文件管理器:
  1. 在文件管理器中打开该目录
  2. 地址栏输入: cmd
  3. 按 Enter

方法 3 - 直接:
  1. 搜索 "命令提示符"
  2. 打开它

进入项目目录:

选择一个命令执行:
