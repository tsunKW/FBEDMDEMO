🔐 安全快速参考
═══════════════════════════════════════════════════════════════════════════════

✓ 推荐做法:
  1. 使用命令行参数: upload.bat <token>
  2. 不要保存 Token 到文件
  3. 不要提交 Token 到 Git
  4. 完成后考虑删除或更新 Token

✗ 避免:
  1. 在脚本中硬编码 Token
  2. 分享或发送 Token
  3. 在公开场所输入 Token
  4. 长期保留同一个 Token

⚠️ 如果 Token 泄露:
  1. 立即访问 https://github.com/settings/tokens
  2. 删除泄露的 Token
  3. 生成新的 Token
